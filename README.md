# quick-websocket-demo
> [!NOTE]
> This is a quick Tic-Tac-Toe game made with WebSocket and SQLite support. A better implementation can definitely be made, but wanted to show my knowledge on WebSockets especially.

> [!IMPORTANT]
> If you want to test the WebSocket features, then:
> 1. Download the code.
> 2. Open the DevContainer. (make sure you have something, like Docker, started)
> 3. Once loaded, type "node server.js" in the Terminal.
> 4. Open a 2nd instance by going on VSCode > Ports, and find the Node.JS port and click the "Open in Browser" icon
> <img width="495" height="120" alt="screenshot" src="https://github.com/user-attachments/assets/c078ac2f-648a-4489-8cd9-18e7780a6ab9" />
> Now, add both of your names, press "Connect" on each instance, and you should connect successfully!
